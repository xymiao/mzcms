# mzcms
基于Java的CMS项目


### 项目结构  
**cms-admin** 后端接口  
**cms-admin-web** 后端管理端页面 前后端分离， 使用vue3 + element-plus  
**cms-common** 存放一些封装的工具之类的  
**cms-pojo** 存在实体类  
**cms-mapper** 数据持久层， 使用mybatis 和 mybatis-plus  
**cms-service**  cms业务相关的模块  
**docs** 项目相关文档存在在该目录下  

### 项目技术栈说明
Spring Boot 2.5.4 全家桶  
Mybatis 3.5.7  
MybatisPlus 3.4.3.3  
Vue3   
ElementPlus
