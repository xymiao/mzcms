# MzCMS 内容管理系统 CMS
## 当前版本

## 技术选型
基于 Java 语言的 Spring 全家桶的内容管理系统。  
**后端技术**  
编程语言： Java
框架选择：
- Spring Boot 
- Mybatis 
- Spring Security

**前端技术**  
后台管理使用分离技术，前端展示暂时使用 thymeleaf  
- Vue 3  
- ElementPlus  
- VueRouter 4  
- Vuex 4  
- axios

所有的技术会随时对应的升级进行升级。升级的过程中有任何的问题，会直接解决掉。 小版本的多次更新比跨大版本的更新更加容易。
